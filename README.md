Технические требования
Системные требования:
Python: 3.12.7 и ниже 
ОС: Windows (тестировалось на Windows)
Камера: Любая USB/встроенная веб-камера
Память: Минимум 4GB RAM для обработки видео

Запуск осуществляется с  командной строки Windows и делается он в 4 Режимах:
1.Детекция коров:
python camtest.py human 
2.Детекция только коров (AI):
pythin camtest.py track
3.Комбинированная детекция с переключением:
python camtest.py both 
4.Одновременная детекция людей и коров:
python camtest.py both_with_human
